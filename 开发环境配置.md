# 简易航运公司管理系统开发环境配置

## 1. 技术栈

本项目采用前后端分离开发方式。

| 层级 | 技术 |
| --- | --- |
| 前端 | Vue 3 |
| 后端 | Java + Spring Boot |
| 数据库 | MySQL |
| 接口风格 | RESTful API |
| 数据格式 | JSON |

前端包管理工具可使用 npm 或 pnpm。后续开发中优先使用 pnpm。

## 2. MySQL 本地环境

当前 MySQL 通过 Scoop 安装。

| 配置项 | 值 |
| --- | --- |
| MySQL 版本 | 9.7.0 |
| 数据库名 | `shipping_management` |
| 主机 | `localhost` |
| 端口 | `3306` |
| 用户名 | `gavin` |
| 密码 | `456123` |

JDBC 连接地址：

```text
jdbc:mysql://localhost:3306/shipping_management?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
```

Spring Boot 数据源配置可参考：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/shipping_management?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: gavin
    password: 456123
    driver-class-name: com.mysql.cj.jdbc.Driver
```

## 3. MySQL 启动方式

当前终端不是管理员权限，因此没有安装 Windows 服务。开发阶段可以使用当前用户方式启动 MySQL。

启动命令：

```powershell
Start-Process -FilePath 'C:\Users\ganmaojun\scoop\apps\mysql\current\bin\mysqld.exe' -ArgumentList '--defaults-file=C:\Users\ganmaojun\scoop\persist\mysql\my.ini' -WindowStyle Hidden
```

检查是否启动成功：

```powershell
mysqladmin ping
```

如果返回：

```text
mysqld is alive
```

说明 MySQL 已正常运行。

## 4. 数据库初始化

项目提供了可执行 SQL 脚本：

```text
sql/schema.sql
```

执行方式：

```powershell
Get-Content -Raw -LiteralPath 'D:\project\swim\sql\schema.sql' | mysql -ugavin -p456123
```

该脚本会创建：

1. 数据库 `shipping_management`。
2. 角色表 `sys_role`。
3. 用户表 `sys_user`。
4. 船舶表 `ship`。
5. 船员表 `crew_member`。
6. 运输任务表 `transport_order`。
7. 财务结算表 `settlement`。
8. 字典项表 `dictionary_item`。
9. 基础演示数据。
